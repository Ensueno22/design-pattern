
# DesignPattern

코드를 작성할 때 객체 지향 적으로 분리 되어지고 변경에 닫히고 오픈에 열려있는 
객체 지향적인 코드를 작성 할 수 있다.

또한 잘 짜여진 코드는 유지보수에도 훨씬 유리한 점이 있다. </br>
